如何获取 GitHub Token？

  1. 登录您的 GitHub 账号，访问 Settings > Developer settings > Personal access tokens > Tokens (classic) 点击此处直达。
  2. 点击右上角 Generate new token (classic)。
  3. 设置 Note 为 AcctVault Sync，Expiration 选择 No expiration (永不过期)。
  4. 在下方权限列表中，仅勾选 gist (Create gists) 这一项。
  5. 点击底部生成按钮，复制得到的 ghp_xxx... 长串文本填入下方。
操作成功
